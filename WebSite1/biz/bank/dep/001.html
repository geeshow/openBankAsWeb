showZoom - 돋보기버튼 생성 예제. ※생성된 돋보기 버튼의 ID는 접미어 '_ZOOM'가 추가됨. 'sample1_ZOOM'. onZoomClick 속성으로 onclick 이벤트 설정가능.
showZoom 샘플1 돋보기 버튼 미적용. 일반적인 text 박스.
showZoom 샘플2 돋보기 버튼 적용 ex) <input type="text" showZoom="after"
showZoom 샘플3 돋보기 버튼 적용 ex) <input type="text" showZoom="before"
onZoomClick 속성으로 onclick 처리를 할 수 있다.
showCalendar - 달력버튼 생성 예제. ※ 생성된 달력 버튼의 ID는 접미어 '_CAL'가 추가됨. 'sample1_CAL'. 클릭시 달력 입력 상자 활성
showCalendar 샘플1 달력 버튼 적용 ex) <input type="text" showCalendar="after"
showCalendar 샘플2 달력 버튼 적용 ex) <input type="text" showCalendar="before"
showZoom / showCalendar - 둘다적용. ※ 같은 위치에 적용시 showZoom 버튼이 밖으로 적용 됨.
둘다적용 샘플1 showCalendar="after" showZoom="after"
둘다적용 샘플2 showCalendar="after" showZoom="after"
둘다적용 샘플3 showCalendar="after" showZoom="after"
suffix - 접미어 ※ input 객체의 접미어로 단위나 속성 등을 표시. ex(원, %, $).
접미어 샘플1 suffix="원"
접미어 샘플2 suffix="%"
접미어 샘플3 suffix="긴접미어"
